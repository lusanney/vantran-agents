# Tool Usage Rules

## GitHub CLI

Do NOT use `gh` CLI on Coder devboxes — it is not installed. Use otter MCP tools instead:
- `mcp__otter__get_pull_request` instead of `gh pr view`
- `mcp__otter__get_pull_request_diff` instead of `gh pr diff`
- `mcp__otter__create_pull_request` instead of `gh pr create`
- `mcp__otter__list_pull_requests` instead of `gh pr list`

## Slack

Never send Slack messages without explicit user approval. Present the draft message and wait.
